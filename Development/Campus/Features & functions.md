

Requirements
- Must work on mobile and desktop and be an inspiring experience regardless of the platform
- Fast to create templates for customers (AI an option)
- Baseline to be used for webapps and social gaming

Features
- Clients should be able to upload and manage content without having to be able to purchase expensive 3D video/images or being able to create special assets for this
- Content should be manageable from Umbraco or other CMS
- Cool page/image/asset transitions
- Less static than the current campus - give reason to re-visit the page often
- Data/analytics features
- Mini-games or experiences with highscore etc
- Personal dashboard / experience-flow
	- Add favorites
	- Choose learning paths
	- User profile


Templates
- Image slider/gallery with transitions
- Presentation tool (ppt in a browser)
	- Live or static
- Videoplayer
- overlays
- navigations


Integrations
- Online meetings
- Streaming video
- AI chats or avatar 
- LMS
