


[[Tools]]
This document contains relevant information about which tools have been used during development and testing of a local setup.

[[Knowledge Graph]]
Knowledge graphs can be used to make meaning of relationships between information. It can be used by the LLM as extra data, when trying to find relevant information to a prompt.

[[RAG]]
The RAG (Retrieval Augmented Generation) is used for storing new data which is accessible to the LLM. 

[[LLM]]
Various LLMs have been tested for local installation.
