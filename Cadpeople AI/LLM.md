Requirements to the LLM used for the test, was to both being able to answer questions but also reply with a structured JSON object with a specific format. This requires that the LLM is able to provide the response with no extra data, as is often best on models, which have been trained  specifically on JSON data.

Pretrained model (foundation model)
Generic training on a large amount of unlabelled data. This is a huge mix of books, Wikipedia, articles, code etc.

Finetuning
Adding specific and labelled data to the dataset to enable the LLM to obtain domain-specific understanding.

This could be a huge amount of e-mails where the spam-mails are tagged and categorized to create a spam-filter AI agent.
